# ShuFengliu (ShuFengliu) - UI设计文档

## 1. 设计原则

### 1.1 视觉风格

- **主题**: 中国古代战争/水墨国风/青铜器质感
- **色调**: 玄黑(#1a1a1a)、朱砂红(#c9372c)、青铜绿(#4a7c59)、宣纸黄(#f5f5dc)
- **氛围**: 肃杀、策略、历史厚重感
- **设计元素**: 竹简、印章、青铜纹饰、山水画

### 1.2 设计目标

1. 清晰展示军阵布局和兵力输送路径
2. 直观呈现将领信息和属性
3. 简洁高效的战前布阵操作
4. 沉浸式的战场氛围

## 3. 界面详细设计

### 3.1 顶部状态栏

```
┌─────────────────────────────────────────────────────────────────────┐
│ [印]  第2轮 / 第3轮    主营 ❤️ 45  vs  ❤️ 72    [设置] [菜单]    │
└─────────────────────────────────────────────────────────────────────┘
```

**元素说明**:
| 元素 | 尺寸 | 说明 |
|------|------|------|
| 印章Logo | 40×40 | 篆书"ShuFengliu"印章 |
| 回合信息 | 200×40 | 当前回合/总回合，竹简样式背景 |
| 主营生命 | 150×40 | 主营血量，带城墙血条 |
| 设置按钮 | 40×40 | 青铜纹饰齿轮图标 |
| 菜单按钮 | 40×40 | 汉堡菜单，竹简样式 |

**生命值显示**:
- 使用城墙图标 + 数字
- 低血量(<30%)时城墙着火动画
- 受伤时城墙崩塌碎片效果

**单槽尺寸**: 120×180 px

```

│  [将领姓名]     │  ← 16px 书法字体

│                 │
│   [将领立绘]    │  ← 80×100 国风立绘

├─────────────────┤

├─────────────────┤
```

**状态指示**:
- 可行动: 金色边框脉冲动画
- 选中: 朱砂红边框 + 光晕
- 受伤: 血色溅射效果
- 被俘/阵亡: 灰色半透明，裂纹覆盖

#### 3.2.3 军资输送可视化

**输送路径动画**:

```
军资从先锋输送到中军:

先锋 [🚚粮草] ──────→ [📜军令] 中军
        ╲        /
         ╲ 路径 /
          ╲    /
           ╲  /
```

**动画效果**:
- 军资: 粮草车、信使快马、军令卷轴
- 路径: 虚线箭头，烽烟轨迹
- 成功送达: 金色波纹扩散，印章"已收"
- 输送损耗: 粮草洒落，数字减少
- 被截击: 伏兵突袭，刀光剑影
- 同阵转交: 军令传递，令旗挥舞

#### 3.3.2 拖放交互

**拖动将领**:
1. 从手牌区拖拽将领卡
2. 军阵槽位高亮显示可放置区域
3. 拖放到目标槽位，播放放置动画（印章"任命"）
4. 槽位已满时显示红色禁止标志

**右键菜单**:
- 查看将领详情
- 快速替换
- 收回手牌

## 4. 视觉元素规范

### 4.1 色彩系统

**主色调**:
```css
--bg-primary: #1a1a1a      /* 玄黑背景 */
--bg-secondary: #2d2d2d    /* 次级背景 */
--bg-card: #3d3d3d         /* 卡片背景 */
--accent-red: #c9372c      /* 朱砂红 */
--accent-green: #4a7c59    /* 青铜绿 */
--accent-gold: #d4a574     /* 古铜金 */
--paper: #f5f5dc           /* 宣纸黄 */
--danger: #8b0000          /* 暗红(危险) */
--success: #228b22         /* 深绿(成功) */
--warning: #b8860b         /* 暗金(警告) */
--text-primary: #f5f5dc    /* 主文字 */
--text-secondary: #a0a0a0  /* 次级文字 */
```

**阵营色**:
- 我方: 青铜绿渐变
- 敌方: 朱砂红渐变
- 中性: 宣纸黄

### 4.2 字体规范

**中文字体**:
- 标题: 方正清刻本悦宋 / 思源宋体
- 正文: 思源黑体 Regular
- 书法: 方正字迹-吕建德行楷
- 数字: DIN / Roboto Mono

**字号**:
| 用途 | 大小 | 字重 |
|------|------|------|
| 大标题 | 32px | Bold |
| 小标题 | 24px | Bold |
| 正文 | 16px | Regular |
| 辅助文字 | 12px | Regular |
| 数值 | 20px | Bold |

### 4.3 图标系统

**属性图标** (24×24):
- ❤️ 生命值 (心形/盾牌)
- ⚔️ 武勇 (交叉剑)
- 🐴 调度 (马蹄/旗帜)
- 🛡️ 接应 (盾牌)
- 👁️ 洞察 (眼睛)

**功能图标** (32×32):
- 设置 (青铜齿轮)
- 菜单 (竹简)
- 确认 (印章"准")
- 取消 (印章"否")
- 信息 (令牌)

## 5. 动画规范

### 5.1 过渡时间

| 动画类型 | 时长 | 缓动函数 |
|---------|------|---------|
| 界面切换 | 300ms | ease-out |
| 将领选中 | 200ms | ease-in-out |
| 军资输送 | 800ms | linear |
| 伤害数字 | 1000ms | ease-out |
| 脉冲效果 | 1500ms | ease-in-out (循环) |

### 5.2 特效规范

**发光效果**:
```css
/* 选中发光 */
box-shadow: 0 0 20px var(--accent-gold),
            0 0 40px var(--accent-gold);

/* 危险发光 */
box-shadow: 0 0 20px var(--danger),
            0 0 40px var(--danger);
```

**粒子效果**:
- 军资输送: 尘土飞扬，马蹄印
- 截击特效: 刀光剑影，血花四溅
- 将领阵亡: 灰化消散，"阵亡"印章落下

## 6. 响应式设计

### 6.1 桌面端 (≥1920×1080)

- 完整4军阵布局
- 手牌区横向排列
- 将领详情面板常驻

### 6.2 平板端 (1280×720 ~ 1920×1080)

- 军阵布局保持4列
- 手牌区可折叠
- 将领详情面板点击弹出

### 6.3 移动端 (<1280×720)

- 军阵布局保持4列但缩小
- 手牌区改为底部抽屉
- 简化部分视觉效果

## 7. 音效设计 (预留)

### 7.1 UI音效

- 按钮点击: 竹简卷动声
- 选中将领: 剑出鞘声
- 放置将领: 印章盖下声
- 错误操作: 铜锣声

### 7.2 战斗音效

- 军资输送: 马蹄声/车轮声
- 输送成功: 号角声
- 输送损耗: 洒落声
- 截击成功: 刀剑碰撞声
- 造成伤害: 攻城撞击声
