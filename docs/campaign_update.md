# 数风流 - 天下统一战模式

## 新增内容

### 大战略模式（天下统一战）
- **地图系统**: 16个郡县组成的古代中国地图
- **势力系统**: 玩家 + 8个AI势力争夺天下
- **轮回继承**: 统一后进入新轮回，继承将领阵容
- **郡县特产**: 占领不同郡县可获得特产名将

### 游戏流程
1. 选择游戏模式（快速对战 / 天下统一战）
2. 天下统一战：
   - 获得5名初始将领
   - 选择相邻郡县进攻
   - 进行卡牌对战
   - 胜者吞并郡县
   - 统一后进入新轮回

### 操作方式
- **地图界面**: 点击郡县选择 → 点击进攻
- **战斗界面**: 与快速对战相同
- **快捷键**: 
  - ESC - 返回主菜单
  - 空格 - 结束回合
