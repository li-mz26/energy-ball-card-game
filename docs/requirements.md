# 定军山 (Dingjunshan) - 需求文档

> **"得一将者得天下，失一将者失江山"**

## 1. 游戏概述

**游戏名称**: 定军山 / Dingjunshan  
**类型**: 回合制策略卡牌游戏 (CCG + 养成)  
**主题**: 中国古代战争/三国/架空历史  
**目标平台**: PC / 移动端  
**开发语言**: Lua (使用 Love2D 框架)  
**玩家人数**: 2人对抗 (支持本地对战/在线对战)

---

## 2. 核心玩法

### 2.1 战场布局 (军阵体系)

双方各拥有 **11名将领**，分为 **4军阵** 放置在战场上：

```
        [敌方主营]
             ↑
    ┌────┬────┬────┬────┐
    │ 先锋 │ 中军 │ 后军 │ 大营 │  ← 我方 (玩家A)
    ├────┼────┼────┼────┤
    │敌大营│ 敌后军│ 敌中军│ 敌先锋│  ← 敌方 (玩家B)
    └────┴────┴────┴────┘
             ↓
        [我方主营]
```

**交叉对阵规则**:  
战场实际排列为 `先锋-敌大营-中军-敌后军-后军-敌中军-大营-敌先锋` 的交替阵型。

| 军阵 | 位置 | 将领数 | 历史职能 |
|------|------|--------|----------|
| **先锋** | A1 vs B4 | 1 vs 1 | 开路先锋、侦查敌情 |
| **中军** | A2 vs B3 | 3 vs 3 | 主力作战、统帅所在 |
| **后军** | A3 vs B2 | 3 vs 3 | 支援补给、预备队 |
| **大营** | A4 vs B1 | 3 vs 3 | 后勤保障、辎重粮草 |

### 2.2 军资输送机制

#### 2.2.1 进攻流程

当进攻方发动攻势时，**军资**（兵力、粮草、辎重）从 **先锋** 开始，沿以下路径向后方输送：

```
先锋 → 中军 → 后军 → 大营 → 敌方主营(假想主营)
```

每份军资在输送过程中会经历以下考验：

#### 2.2.2 军资输送结果

对于每一次输送，军资可能经历：

1. **成功送达**: 军资到达目标军阵
2. **输送损耗**: 粮草被消耗/兵力逃散/辎重损坏
3. **被截击**: 被路径上的敌方将领截获
4. **同阵转交**: 转移到同军阵的另一将领处，由其继续输送

**同阵转交限制**: 已被转交过的军资 **不会再次被转交**。

### 2.3 输送判定详解

#### 2.3.1 目标选择

输送将领根据以下因素决定目标：
- 可观察到的目标将领状态
- 目标将领的"接应能力"
- 概率分布由输送将领的AI/玩家策略决定

#### 2.3.2 输送损耗率

```
损耗率 = 基础损耗率(25%) 
       × (1 - 输送方调度能力/100) 
       × (1 - 接收方接应能力/100)  
       × (1 + 军阵间距/10)
最小值: 5%, 最大值: 85%
```

影响因素：
- 输送方调度能力越高，损耗率越低（善于行军调度）
- 接收方接应能力越高，损耗率越低（善于整顿接收）
- 军阵间距越远，损耗率越高（长途输送损耗大）

#### 2.3.3 截击机制

当军资从军阵X输送到军阵Y时：
1. 系统计算输送路径
2. 找出距离路径 **最近** 的敌方将领
3. 该敌方将领执行截击判定

```
截击成功率 = 基础截击率(35%)
           × (截击者武勇/100)
           × (1 - 距离到路径/5)
最小值: 5%, 最大值: 90%
```

- 截击者武勇越高，截击成功率越高
- 截击者距离路径越近，截击成功率越高

## 3. 将领系统

### 3.1 将领属性

每位将领拥有以下基础属性：

| 属性 | 说明 | 影响机制 |
|------|------|----------|
| **姓名** | 将领名称 | 历史名将或随机生成 |
| **品阶** | 将领稀有度 | 白/绿/蓝/紫/橙 |
| **朝代** | 所属朝代 | 羁绊加成 |
| **姓氏** | 家族姓氏 | 羁绊加成 |
| **祖籍** | 出生地/籍贯 | 羁绊加成 |
| **生命值** | 将领耐久 | 被截击/反击时损失 |
| **武勇** | 个人武力 | 影响截击成功率 |
| **调度** | 行军调度 | 影响输送损耗率 |
| **接应** | 整顿接应 | 影响接收损耗率 |
| **洞察** | 战场洞察 | 可观察的目标范围 |
| **特技** | 专属技能 | 历史典故还原 |

### 3.2 羁绊系统 (新增)

将领之间存在三种羁绊关系，影响军资输送成功率：

#### 3.2.1 朝代羁绊
- **同朝传送**: 输送成功率 +15%
- 示例: 蜀汉将领之间互相输送

#### 3.2.2 姓氏羁绊  
- **同姓传送**: 输送成功率 +10%
- 示例: 张家将领之间（张飞、张辽等）
- **同宗 bonus**: 若同时同朝同姓，额外 +5%

#### 3.2.3 祖籍羁绊
- **同乡传送**: 输送成功率 +10%
- 示例: 同为并州出身的将领
- **地缘 network**: 若输送链上连续3人同祖籍，第三人起额外 +5%

#### 3.2.4 羁绊叠加规则
```
基础成功率: 70%
+ 同朝: +15%
+ 同姓: +10%  
+ 同籍: +10%
+ 同朝同姓 bonus: +5%
最大成功率: 95%
```

羁绊效果在布阵阶段可视化显示（连线高亮）。

### 3.3 将领获取与养成

#### 3.3.1 将领获取

- **招募**: 消耗资源从卡池抽取
- **战役掉落**: 完成单人战役获得特定将领
- **掠夺**: 战胜对手后掠夺对方将领 (见第5章)

#### 3.3.2 将领升级

```
升级消耗 = 基础消耗 × (当前等级^1.5)
```

升级提升：
- 生命值 +10%
- 主属性 (武勇/调度/接应) +5%
- 每5级解锁/强化特技

#### 3.3.3 将领进阶 (突破)

- 消耗同名将领碎片进行进阶
- 进阶提升品阶，解锁新特技
- 最高可升至红色品质

#### 3.3.4 装备系统

每位将领可装备：
- **武器**: 提升武勇
- **坐骑**: 提升调度
- **兵法**: 提升接应
- **宝物**: 特殊效果

## 4. 牌组与军阵

### 4.1 牌组构建

- 玩家构建自己的将领牌组（11张将领卡）
- 根据品阶有总费用限制
- 相同将领最多携带2张

### 4.2 战前布阵

每轮进攻前，进攻方可以重新布阵：

1. 从牌组中选择将领
2. 放置到4个军阵中（先锋1人，其余各3人）
3. 考虑对阵关系和将领克制
4. 确认布阵后不可更改

---

## 5. 掠夺机制 (高风险高回报)

### 5.1 掠夺规则

**对战结束后**: 
- **胜利方**: 可从败方牌组中选择 **1张将领卡** 掠夺
- **败方**: 失去被掠夺的将领，进入"俘虏营"，24小时后可赎回

### 5.2 掠夺限制

- 每场对战只能掠夺1张
- 无法掠夺品阶高于自己最高品阶的将领
- 新获得将领（24小时内）无法被掠夺
- 每周最多被掠夺3次，超过后获得保护

### 5.3 赎回机制

被掠夺的将领：
- 进入胜方的"俘虏营"显示24小时
- 败方可支付资源提前赎回
- 24小时后自动释放回原主人
- 胜方在此期间可使用该将领体验

### 5.4 心理设计

- 掠夺时显示将领的养成信息（等级、强化次数）
- 被掠夺时有视觉和音效反馈（"X将军被俘！"）
- 败方可选择"复仇"快速匹配再次对战

## 6. 游戏回合

### 6.1 回合结构

游戏共进行 **3轮**，每轮包含：

1. **布阵阶段** (进攻方)
   - 可调整军阵配置
   - 从牌组选择将领放置

2. **进攻阶段** (进攻方)
   - 从先锋发动军资输送
   - 沿链条传递
   - 最终伤害结算

3. **防守阶段** (防守方)
   - 被动承受攻击
   - 截击机会

### 6.2 回合交替

```
第1轮: A进攻 → B防守 → B进攻 → A防守
第2轮: A进攻 → B防守 → B进攻 → A防守  
第3轮: A进攻 → B防守 → B进攻 → A防守
```

### 6.3 胜利条件

- **即时胜利**: 敌方主营生命值归零
- **最终胜利**: 3轮结束后，生命值较高的一方获胜

## 7. 数值系统

### 7.1 基础公式

**输送损耗率**:
```
损耗率 = 基础损耗率(25%) 
       × (1 - 输送方调度/100) 
       × (1 - 接收方接应/100)  
       × (1 + 军阵间距/10)
最小值: 5%, 最大值: 85%
```

**截击成功率**:
```
截击率 = 基础截击率(35%)
       × (截击者武勇/100)
       × (1 - 距离到路径/5)
最小值: 5%, 最大值: 90%
```

**伤害计算**:
```
伤害 = 到达主营的军资份数 × 单份伤害(10点)
```

| 紫 | 50 | 2 | 48小时 |

## 8. 游戏模式

### 8.1 教学模式

- 逐步引导玩家了解军阵体系
- 历史典故讲解

### 8.2 本地对战

- 两名玩家在同一设备上对战
- 热座模式

### 8.3 人机对战

- 玩家 vs AI (历史名将AI)
- 多个难度等级

### 8.4 排位对战 (核心玩法)

- 匹配真实玩家
- 赛季制，根据胜负调整段位
- **掠夺机制仅在排位生效**

### 8.5 单人战役 (未来扩展)

- 还原历史战役
- 获得特定将领
