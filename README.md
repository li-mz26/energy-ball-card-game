# 定军山 (Dingjunshan)

> **"得一将者得天下，失一将者失江山"**

基于 Lua/Love2D 开发的回合制策略卡牌游戏，融合中国古代战争背景与高风险掠夺机制。

## 游戏概念

双方各控制11名历史将领，分 **先锋、中军、后军、大营** 四军阵进行交叉对阵。通过**军资输送链**（兵力、粮草、辎重）攻击敌方主营，3轮对决决出胜负。

**核心特色：掠夺机制** — 胜利方可收编敌方将领，失败者将失去辛苦培养的卡牌。

## 战场布局

```
        [敌方主营]
             ↑
    ┌────┬────┬────┬────┐
    │ 先锋 │ 中军 │ 后军 │ 大营 │  ← 我方
    ├────┼────┼────┼────┤
    │敌大营│ 敌后军│ 敌中军│ 敌先锋│  ← 敌方
    └────┴────┴────┴────┘
             ↓
        [我方主营]
```

## 核心机制

### 军资输送链
军资从**先锋**开始，沿 `先锋 → 中军 → 后军 → 大营 → 敌方主营` 输送。过程中可能发生：
- ✅ **成功送达** — 军资抵达目标军阵
- 💨 **输送损耗** — 粮草被消耗、兵力逃散
- ⚔️ **被截击** — 被路径上的敌方将领截获
- 🔄 **同阵转交** — 转移到同军阵的另一将领处继续输送

### 将领养成
- **升级**: 提升生命值和主属性
- **进阶**: 消耗同名碎片提升品阶，解锁新特技
- **装备**: 武器、坐骑、兵法、宝物

### 掠夺机制
- **胜利掠夺**: 战胜后可从败方牌组中选择 **1张将领卡** 收编
- **俘虏系统**: 被掠夺将领进入"俘虏营"24小时，期间胜方可使用
- **赎回机制**: 败方可支付资源提前赎回

### 羁绊系统
将领之间存在三种羁绊关系，影响军资输送成功率：
- **同朝羁绊**: 相同朝代的将领间输送 +15%成功率
- **同姓羁绊**: 相同姓氏的将领间输送 +10%成功率  
- **同籍羁绊**: 相同祖籍的将领间输送 +10%成功率
- **叠加奖励**: 同朝同姓额外 +5%

## 将领属性

每位将领拥有：
- **基础**: 姓名、品阶(白/绿/蓝/紫/橙)、生命值
- **羁绊**: 朝代、姓氏、祖籍
- **能力**: 武勇、调度、接应、洞察
- **特技**: 历史典故还原的技能

## 技术栈

- **引擎**: Love2D
- **语言**: Lua 5.1+
- **分辨率**: 1920×1080 (可缩放)
- **风格**: 水墨国风 + 青铜器质感
