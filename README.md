# 能量球 (Energy Ball)

基于 Lua/Love2D 开发的回合制策略卡牌游戏。

## 游戏概念

双方各控制11个单位，分成4排进行交叉对阵。通过能量球的传递链条攻击敌方大本营，3轮对决决出胜负。

## 核心机制

### 战场布局
```
        [敌方大本营]
             ↑
    ┌────┬────┬────┬────┐
    │ A1 │ A2 │ A3 │ A4 │  ← 我方
    ├────┼────┼────┼────┤
    │ B4 │ B3 │ B2 │ B1 │  ← 敌方
    └────┴────┴────┴────┘
             ↓
        [我方大本营]
```

### 能量球传递
能量球从A1开始，沿 `A1 → A2 → A3 → A4 → 敌方大本营` 传递。过程中可能发生：
- ✅ 成功发送
- ❌ 发送失误
- 🛡️ 被拦截
- 🔄 同排转移
